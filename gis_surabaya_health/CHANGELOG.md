# Changelog - Frontend GIS Surabaya Health

Semua perubahan pada projek frontend akan dicatat di file ini.

## [1.0.0] - 2026-05-13
### Added
- Inisialisasi projek React dengan TypeScript menggunakan Vite.
- Integrasi Tailwind CSS untuk styling modern.
- Implementasi Map Component menggunakan `react-leaflet`.
- Implementasi Dashboard Page untuk ringkasan statistik.
- Konfigurasi Routing menggunakan `react-router-dom`.
- Integrasi API menggunakan Axios untuk mengambil data dari backend.
- Set up Typescript interfaces untuk data GIS.
